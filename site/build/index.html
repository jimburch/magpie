Sharing AI workflows is still a mess

Someone posts their Claude Code setup on Twitter. It looks amazing. But recreating it means hunting through screenshots, copying config snippets, manually installing MCP servers, and hoping you didn't miss anything.

Dotfiles repos help, but they weren't built for AI tooling. They don't capture MCP configs, custom skills, project-specific prompts, or the context that makes a workflow actually work.

One command. Entire workflow.

Coati packages everything — configs, scripts, MCP servers, skills, docs — into a single shareable setup. Clone any developer's complete workflow instantly.

Terminal

$ coati clone jsmith/typescript-fullstack

✓ Cloned setup.json

✓ Installed 3 MCP servers

✓ Applied .claude/settings.json

✓ Copied CLAUDE.md and .cursorrules

Done! Your workspace is ready.

How it works

Clone
